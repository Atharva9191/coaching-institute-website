# Bright Future Coaching Website

A modern coaching institute website created using HTML and CSS.  
This project includes a homepage, courses page, and admission form with a clean and professional design.

## Features

- Modern responsive layout
- Hero landing section
- Buy One Get One promotional section
- Courses table and cards
- Admission form
- Footer with contact details
- Smooth hover animations
- Professional UI design

## Pages

1. Home Page
2. Courses Page
3. Admission Page

## Technologies Used

HTML5  
CSS3  
Flexbox  
CSS Grid  
Animations

## How to Run the Project

1. Download or clone the repository
2. Open the folder
3. Double click `index.html`

The website will open in your browser.
